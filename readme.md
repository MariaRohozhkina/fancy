Сайт c прогнозом погоды. Реализован поиск по карте, голосовой ввод. Также работа с weather API. Картинка на background получается с помощью unsplash API с учетом текущего сезона, времени суток и погоды. Градусы можно смотреть и по Фаренгейту, и по Цельсию. 
В проекте помимо этого используется webpack, eslint. Верстка респонсив.
